HTML5

HTML5 é a versão mais recente da linguagem de marcação de hipertexto (Hyper Text Markup Language) utilizada para criar e estruturar conteúdo na web. Ele traz diversas melhorias e novas funcionalidades em relação às versões anteriores, facilitando a criação de páginas web mais interativas, responsivas e com conteúdo multimídia.

Características:

  • É uma linguagem de marcação
  • É uma linguagem interpretada
  • Roda no lado do cliente
  • Não é case-sensitive
  • É formada por tags

CSS

CSS, ou Folhas de Estilo em Cascata, é uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Simplificando, o CSS é responsável pela aparência de um site, determinando como cores, fontes, layout e outros elementos visuais são exibidos.

Características:

  • É uma linguagem de formatação e estilização visual
  • É uma linguagem interpretada
  • Roda no lado do cliente
  • É case-sensitive
  • É formada por seletores e propriedades
